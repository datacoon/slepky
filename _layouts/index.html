Готовые базы данных
по регионам и муниципалитетам РФ

Источник подготовленных «слепков» баз данных, содержащих информацию по государственным закупкам и юридическим лицам регионов и муниципалитетов РФ. Используется в работе аналитиками, чиновниками, специалистами по проверке контрагентов и участниками разработки стратегий развития регионов.

Базы данных по государственным закупкам

Данные по контрактам, закупкам, протоколам и пр. по 44-ФЗ и 223-ФЗ, относящиеся к конкретному региону или муниципалитету.

Стоимость баз данных

Цена в значительной мере зависит от того, какой именно регион / муниципалитет выбирает клиент, поэтому предварительно мы предлагаем ориентировочные цены. Конкретная цена определяется после оформления заявки с указанием точных параметров запроса

Базы данных по государственным закупкам

Ориентировочная цена за базу данных по региону РФ (кроме Москвы) по одной сущности (к примеру, контракты по 44-ФЗ) за один год (за 2017) – 10 000 рублей.

При возрастании количества требуемых периодов цена возрастает незначительно (1-2 000 рублей за каждый год).

При возрастании количества требуемых сущностей цена возрастает примерно на 50% первоначальной стоимости (5 000 рублей за дополнительную сущность).

Базы данных по данным юридических лиц (ЕГРЮЛ)

Ориентировочная цена за базу данных по региону РФ (кроме Москвы) – 10 000 рублей.

Базы данных по бухгалтерской отчетности юридических лиц (Росстат)

Ориентировочная цена за базу данных по региону РФ (кроме Москвы) за один год (за 2017) – 10 000 рублей.

При возрастании количества требуемых периодов цена возрастает незначительно (1-2 000 рублей за каждый год).
